Rust + ratatui + crossterm

VCMC

Быстрый двухпанельный файловый менеджер в стиле Midnight Commander. Локальный режим, SFTP, архивы, viewer, editor и поиск прямо из терминала.

Что уже есть

  • 2 панели, асинхронные операции, responsive UI без фризов.
  • SFTP-подключения и bookmarks (add / edit / delete / connect).
  • Поиск файлов через fd и по содержимому через rg.
  • Viewer (text/hex, поиск) и внешний editor по F4.
  • Archive VFS для zip/tar/tgz (read-only browse + copy out).

Быстрые клавиши

Tab панели · F5/F6/F8 copy/move/delete · F9 меню · F10 выход

F3 viewer · F4 editor · / поиск · Ctrl+O shell

Почему так

Для 1-страничника самый простой и быстрый путь: один index.html без сборки и зависимостей.

Когда нужен MkDocs: если хочешь много страниц документации, навигацию, поиск и автопубликацию docs.
