# Step 5: Навигация и управление фокусом

## Цель
Довести навигационное поведение до привычного двухпанельного менеджера: стабильные хоткеи, корректный переход по `..`, быстрый jump в home и отсутствие рассогласований состояния между панелями.

## Решения
- Добавить виртуальный элемент `..` в начало листинга (кроме корня).
- Обрабатывать `Enter` по `..` так же, как переход в родительский каталог.
- Защитить операции `Copy/Move/Delete` от запуска на виртуальных навигационных элементах.
- Расширить hotkeys для home (`Home` и `~`).
- Нормализовать статус/выделение панели после reload без побочных эффектов для второй панели.
