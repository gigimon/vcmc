# Step 7: Поиск и сортировка

## Цель
Добавить инкрементальный поиск по имени в текущем каталоге и закрепить сортировку как независимую настройку каждой панели.

## Решения
- Ввести на уровне панели:
  - `all_entries` (полный листинг),
  - `entries` (отфильтрованный видимый список),
  - `search_query`.
- Добавить режим поиска:
  - `/` включает поиск на активной панели;
  - ввод символов сразу фильтрует видимый список;
  - `Backspace` удаляет символ;
  - `Enter` завершает режим с сохранением фильтра;
  - `Esc` сбрасывает фильтр.
- Сортировка остается `per-panel` через `sort_mode` активной панели (`F2`).
- Фильтрация выполняется в памяти по `all_entries`, без дополнительных FS-запросов.
