# Step 10: Rename Dialogs и Табличный UI

## Цель
Добавить управляемое окно изменения имени для операций `copy/move` и сделать более информативный список файлов в панелях.

## Решения
- `F5/F6` теперь открывают rename-диалог с предзаполненным именем выбранного элемента.
- После подтверждения (`Enter`) операция уходит в job queue с целевым именем в неактивной панели.
- Добавлены правила валидации имени (`не пустое`, без `/`) и отмена по `Esc`.
- Панели отображаются в 3 колонках:
  - `Name`
  - `Size`
  - `Modified`
- Цвета типов:
  - директории — синий,
  - симлинки — magenta,
  - навигационный элемент `..` — желтый.
