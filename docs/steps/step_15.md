# Step 15: Нижнее меню F-key в стиле MC

## Цель
Заменить нижнюю строку подсказок на визуальные кнопки в стиле Midnight Commander и сделать их контекстно-зависимыми.

## Решения
- В `ui.rs` текстовый `render_help` заменен на `render_footer`.
- Footer теперь показывает режим экрана:
  - `[NORMAL]` при обычной навигации
  - `[SELECTION]` при наличии выделения
  - `[DIALOG]` при открытом диалоге
- Для `Normal/Selection` добавлены визуальные кнопки `F1..F10`:
  - активные действия (`F5/F6/F8` и др.)
  - disabled-заглушки для еще не реализованных (`F1/F3/F4/F9`)
- Для `Dialog` показывается отдельный набор управления:
  - навигация фокуса (`Tab`, `Shift+Tab`, `Left/Right`)
  - кнопки текущего диалога (`Alt+Y/Alt+N`, `Alt+A/Alt+C`, `OK` и т.д.)
- Добавлена визуальная дифференциация состояний:
  - `active`: яркая подсветка
  - `enabled`: стандартный темный фон
  - `disabled`: приглушенный цвет
- В `app.rs` добавлен `F10` как горячая клавиша выхода для соответствия footer.
