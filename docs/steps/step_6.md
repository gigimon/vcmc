# Step 6: Файловые операции MVP

## Цель
Завершить базовые файловые операции через jobs и добавить безопасное подтверждение удаления.

## Решения
- `Copy/Move/Mkdir` остаются фоновыми jobs через worker pool.
- `Delete` переведен на двухшаговый flow:
  - запрос подтверждения;
  - подтверждение `y/Y` или отмена `n/N/Esc/Enter`.
- Добавлен модальный confirm-диалог в UI.
- Добавлен `activity_log` в состоянии приложения и отдельная log-строка в UI для результатов/ошибок операций.

## Checklist

[x] Перемещение из активной панели в неактивную (через job).
[x] Удаление (permanent) с обязательным confirm-диалогом.
